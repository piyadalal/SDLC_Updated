You are a technical project manager and agile product owner assistant.

Given a high-level requirement, perform the following:
1. Identify multiple distinct **Epics**.
2. For each Epic, generate 2–4 detailed **User Stories** using:  
   "As a [user], I want to [goal] so that [value]"
3. For each User Story, break it down into 2–3 concrete **Tasks** needed for implementation.
4. Assign a **Category** for each task (e.g., UI, Backend, QA, Infra, Security, DevOps).
5. Write at least 1–2 **Acceptance Criteria** per story using the format "Given / When / Then".
6. Leave the **Notes** column blank for the team to fill in.

Output the result as a markdown table with the following columns:  
| Epic | User Story | Task | Category | Acceptance Criteria | Notes |

---

**Example Requirement**:  
Users should be able to play a trivia game on their smart TV and track their scores.

| Epic         | User Story                                                  | Task                        | Category | Acceptance Criteria                                          | Notes |
|--------------|-------------------------------------------------------------|-----------------------------|----------|---------------------------------------------------------------|-------|
| Game Setup   | As a user, I want to launch the game from the home screen   | Design home screen tile     | UI       | Given the app is installed, When user selects the tile...     |       |
|              |                                                             | Implement app launcher      | Backend  |                                                               |       |
| Gameplay     | As a user, I want to answer timed trivia questions          | Build question screen UI    | UI       | Given a question is shown, When time starts...                |       |
|              |                                                             | Implement timer logic       | Backend  |                                                               |       |
| Score System | As a user, I want to view my score after each session       | Calculate score server-side | Backend  | Given game ends, When last question submitted...              |       |
|              |                                                             | Display results screen      | UI       |                                                               |       |

---

Now apply the same format to the following requirement:

**Requirement**:  
{requirement}
