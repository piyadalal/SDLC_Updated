Title: Simple Trivia Quiz Game for TV Platform

Overview:
Create a single-player trivia game application for Smart TVs that allows users to answer multiple-choice questions using the remote control. The app should be easy to navigate, visually clean, and suitable for casual play.

Quiz Screen

Highlight selected option

Option to play again and choose a different category

Question Data

Served from a backend API endpoint

API should support category-based question retrieval

Each category must support at least 10 questions

Responsive to different TV resolutions
